Java
====

Install Java JREs and optionally JDKs.


Role Variables
--------------

Optional variables:
- `java_jre_versions`: A list of JRE versions to install, default `[1.8.0]`
- `java_jdk_install`: If `True` install JDKs corresponding to the JRE versions, default `False`
