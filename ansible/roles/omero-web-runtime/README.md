OMERO Web Runtime
=================

OMERO.web runtime dependencies.

Mostly Python modules.

Dependencies
------------

Depends on basedeps, ice, optionally redis.


Role Variables
--------------

Optional variables:
- `omero_web_runtime_redis`: If `True` install redis dependencies, default `False`
